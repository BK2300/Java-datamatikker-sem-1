- Levels 1-10: Basic Java (variables, loops, conditions, functions)
- Levels 11-20: Object-oriented programming (classes, objects, inheritance, interfaces)
- Levels 21-30: Data structures and algorithms (arrays, lists, stacks, recursion)
- Levels 31-40: Advanced concepts (files, database connections, threads, networks)
- Levels 41-50: Mini-projects and software development (games, APIs, frameworks)
